# Hand Gesture Recognition using SVM

## SkillCraft Technology - Task 4

### Objective
Develop a hand gesture recognition model to classify different hand gestures from image data.

### Algorithm Used
- Support Vector Machine (SVM)

### Tools
- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

### Dataset
Hand Gesture Dataset (gesture_small)

### Process
1. Loaded grayscale hand gesture images.
2. Resized images to 50×50 pixels.
3. Converted images into feature vectors.
4. Split data into training and testing sets.
5. Trained an SVM classifier.
6. Evaluated model performance using accuracy and classification report.
7. Visualized predicted gestures.

### Result
- Dataset Size: 1697 images
- Number of Classes: 10
- Accuracy Achieved: 100%

The model successfully classified hand gesture images and generated correct predictions on the test dataset.
